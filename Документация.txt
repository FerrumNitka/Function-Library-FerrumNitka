=== Про библиотеку ====
Эта библиотека предназначена для работы с терминалом Windows
[ Внимание! это библиотека использует функции из "Windows.h", это библиотека не может
работать с другими ос, только Windows ]

=== Приступая к работе ====
1. Добавьте файлы библиотеки в проект 
2. Подключите заголовочный файл "FunctionLibrary.h"
3. Перед тем как использовать функции из библиотеки, необходимо вызвать функцию init_variables
4. В конце кода нужно вызвать функцию close_variables

пример простой программы:
#include "FunctionLibrary.h"
#include "colors.h"
#include "transparency.h"


int main()
{
	transparency();
	init_variables();

	set_color(color_red, color_gray);
	print_info();

	std::cin.ignore();
	close_variables();
	return 0;
}



=== разбор функций и макроса ====
init_variables();	Инициализирует переменные handle и coord


close_variables();	Вызывает функцию из "Windows.h" CloseHandle(handle)


print_info();	Выводит версию библиотеки


in_parentheses(std::string text, uc type);
type тип
Возвращает текст в скобках, есть два типа
parentheses (Круглые скобки) и square_brackets (Квадратные скобки)


print_line(uc length);	Выводит символ "=" определённое количество раз
length	длина


set_color(uc color_text, uc color_background);	Задаёт цвет текста и фона, перед использованием
рекомендуется подключить заголовочный файл "colors.h"
color_text	цвет текста
color_background	цвет фона текста


print_message(whose_message whose_message, std::string text);	Выводит текст: [whose_message]	text
whose_message	это typedef std::string whose_message; проще говоря строка


transparency(); Убирает прозрачность окна терминала, нужно подключить заголовочный файл "transparency.h"

=== Ошибки и как их решить ====
Не работает set_color: Вы забыли вызвать функцию init_variables
